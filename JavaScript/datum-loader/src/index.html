SolarNode Datum Loader

A demonstration of the Datum Loader SolarNetwork SDK package.

Use the Find Sources button to find all available sources, or enter a source ID wildcard pattern here first to find sources that match that pattern.

There are several helpers demonstrated on this page:

  • The DatumRangeFinder class is used via the Find Range button to fetch the available date range.
  • The DatumSourceFinder class is used via the Find Sources button to fetch the available sources.
  • The DatumLoader class is used via the Load Datum button to fetch the actual data.
